Neste artigo

    O que é Desenvolvimento Web?

    Desenvolvimento Web engloba a criação de aplicações e sites na internet. O front-end cuida do que o usuário vê e interage (HTML, CSS, JavaScript, React). O back-end cuida da lógica do servidor, banco de dados e APIs (Node.js, Python, Java, PHP). O full-stack domina ambos.

    Grade curricular

    HTML5, CSS3, JavaScript nos primeiros módulos. Frameworks front-end (React, Vue ou Angular), back-end com Node.js ou Python, banco de dados (MySQL, PostgreSQL, MongoDB), REST APIs, autenticação e segurança, versionamento Git, deploy em cloud (Vercel, AWS, Heroku).

    Mercado de trabalho

    O mercado de desenvolvimento web é o mais acessível da área de TI. Muitos desenvolvedores conseguem o primeiro emprego ainda durante o curso com portfólio no GitHub. Plataformas de freelance como Toptal, Upwork e 99designs permitem trabalhar para clientes internacionais em dólar desde o início da carreira.
