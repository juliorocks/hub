Resumo rápido

Salário inicial: R$ 3.000-4.500 (junior). 2-5 anos: R$ 5.000-8.000 (pleno). Senior: R$ 10.000-18.000. Demanda: altíssima — startups, agências, bancos e tech companies procuram desesperadamente. Neste artigo

    O que é UX/UI Design (de verdade)

    Muita gente confunde:

    • UX (User Experience): como o usuário se sente usando o produto. Pesquisa, testes, fluxo, dados
    • UI (User Interface): como o produto parece. Cores, tipografia, ícones, botões
    • Realidade: na maioria das empresas, você faz os dois. É raro ter puro UX ou puro UI

    Ferramentas essenciais (aprenda isso primeiro)

    • Figma: padrão da indústria, grátis para começar, colaborativo
    • Adobe XD: alternativa, mais pesada, usado por agências grandes
    • Prototyping: Figma proto nativo, ou Framer para animações avançadas
    • Pesquisa: não é ferramenta, mas habilidade — aprenda a fazer entrevistas, testes de usabilidade

    Onde trabalhar em 2026

    Maior demanda:

    • Fintechs (Nubank, Picpay, Mercado Pago, etc) — mega demanda
    • Startups — crescimento rápido, precisa redesenhar tudo constantemente
    • Agências de publicidade — mudança constante, cliente exigente
    • Bancos (Itaú, Bradesco) — projetos grandes, salários altos, processo seletivo rigoroso
    • Tech companies (Google, Microsoft, Apple offices no BR)
    • Freelancer — melhor controle de agenda, mas precisa de portfolio forte

    Como começar do zero (sem faculdade formal)

    1. Aprenda online: Coursera, Udemy, YouTube — cursos de UX/UI Design (40-60h)
    2. Aprenda Figma: 2-3 semanas de prática focada
    3. Estude casos reais: veja como Uber, Netflix, WhatsApp resolvem problemas de design
    4. Crie portfolio: 3-5 projetos pessoais completos (redesign de apps, novo app, landing page)
    5. Compartilhe: Behance, Dribbble, Twitter — mostre progresso
    6. Aplique para junior: agora com portfolio, você é hirável

    Diferenças: UX Designer vs Designer Gráfico vs Web Designer

    • UX Designer: foca em como usuário interage, dados, pesquisa, apps/produtos
    • Designer Gráfico: foca em estética, publicidade, posters, branding tradicional
    • Web Designer: foca em websites estáticos, menos UX, mais visual

    Qual tem mais demanda? UX Design. Onde cresce mais? Também UX Design.

    Dica de ouro para conseguir o primeiro emprego

    Não procure por "Junior UX Designer". Procure por:

    • Product Designer (entrada comum)
    • UI Designer (se tem mais experiência visual)
    • Design Intern (estágio, caminho legítimo)

    Essas vagas junior na realidade aceitam candidatos com 0-2 anos de experiência com portfolio.
